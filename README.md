Game Controls
--------
  player 1:

    Up Arrow : move Up
  
    Down Arrow : move Down
  
    Right Arrow : move Right
  
    Left Arrow : move Left
  
    SPACE : Attack (Must Hold Direction Key + SPACE to Attack)
  
    i : Toggle Inventory
    
  Inventory:
    w : Scroll Up
    
    s : Scroll Down
